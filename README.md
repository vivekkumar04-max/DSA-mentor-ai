<h1><b>DSA MENTOR AI</b></h1>
<br>
An AI-powered Data Structures & Algorithms mentor built using Next.js and Tambo Cloud.
This project act as your DSA mentor that helps students build problem-solving skills in
<br>
C++ through step-by-step explanation.
<br>
<h2><b>Problem statement</b></h2>
<h3>Student from tier-2 and tier-3 colleges offten struggle with:</h3>
  1. Structured DSA preparation<br>
  2. thinking about the intution<br>
  3. Understanding time & space complexity<br>
  4. Practicing optimized solutions<br>
<br>
<h2><b>Solution</b></h2>
  <h3>DSA Mentor AI:</h3>
    1. Encourage students to think before giving answer<br>
    2. Then explains the brute force approach<br>
    3.  provides code in C++ by default<br>
    4. Explains time and space complexity<br>
    5. AI powered by Tambo Cloud LLM integration<br>
<br>
 <h2><b>Features</b></h2>   
    1. DSA focused Ai mentor<br>
    2. C++ default solutions<br>
    3. Time and Space complexity explanation<br>
    4. Step-by-step problem solution<br>
    5. Cloud-hosted AI via Tambo<br>
